# Postman Files

Import these files into Postman:

1. `paymentservice-apis.postman_collection.json`
2. `paymentservice-apis-v2.postman_collection.json`
3. `paymentservice-local.postman_environment.json`
4. `paymentservice-dev.postman_environment.json`
5. `paymentservice-prod.postman_environment.json`

Then select one environment (`local`, `dev`, or `prod`) and run requests.

Default URLs:
- local: `http://localhost:8085`
- dev: `http://localhost:8085`
- prod: `https://payment.voyagevibes.in`
