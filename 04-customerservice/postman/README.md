# Postman Files

Import these files into Postman:

1. `customerservice-apis.postman_collection.json`
2. `customerservice-apis-v2.postman_collection.json`
3. `customerservice-local.postman_environment.json`
4. `customerservice-dev.postman_environment.json`
5. `customerservice-prod.postman_environment.json`

Then select one environment (`local`, `dev`, or `prod`) and run requests.

Default URLs:
- local: `http://localhost:8084`
- dev: `http://localhost:8084`
- prod: `https://customer.voyagevibes.in`
