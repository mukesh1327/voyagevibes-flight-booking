# Postman Files

Import these files into Postman:

1. `bookingservice-apis.postman_collection.json`
2. `bookingservice-apis-v2.postman_collection.json`
3. `bookingservice-local.postman_environment.json`
4. `bookingservice-dev.postman_environment.json`
5. `bookingservice-prod.postman_environment.json`

Then select one environment (`local`, `dev`, or `prod`) and run requests.

Default URLs:
- local: `http://localhost:8083`
- dev: `http://localhost:8083`
- prod: `https://booking.voyagevibes.in`
